### SpaceShooter / SpaceRocks

## Controls
 - Arrow Up (Move Upwards)
 - Arrow Left (Move Left)
 - Arrow Right (Move Right)
 - Escape (Main Menu)
 - Left Mouse (Shoot)
## Powerups
 - Ghost (Invinvibility)
 - Spread (Dupes your bullets 3x)
